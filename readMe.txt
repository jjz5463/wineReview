This project is created for DS 310 Wine review
there are two juypter notebooks, 2 folders, and one submit file in this folder
- Two folders
	- raw_data: contain the raw dataset provided for this wine review project
	- processed_data: contain the processed datasets after preprocessing

- Two juypter notebooks
	- pre_processing.ipynb: focused on preproessing. Run this file first
	- modeling.ipynb: focused on training. Run this file after pre_processing.ipynb

- submit.csv: actual submit file
